# 🏁 Gatsby Default Theme

A blank theme for Gatsby using

- Styled Components
- Wes Bos No-Sweat™ Eslint and Prettier Setup

### Start developing

```
git clone
```

Navigate into your new site’s directory and start it up.

```
npm install
gatsby develop
```

The site is running at
`http://localhost:8000`
`http://localhost:8000/___graphql`

### Build

```
gatsby build
gatsby serve
```
